https://crontab.guru/ can help you to write your crontab expression. Examples:
*/5 * * * * - execute every 5 minutes
* */1 * * * - execute every hour
0 0 * * * - execute every day at midnight
0 0 1 * * - execute every month on the first day
0 0 * * 2,3 - execute every Tuesday and Wednesday at midnight
